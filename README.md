# peraktikum_4
Fitur Program

Meminta input data mahasiswa berupa: Nama, NIM, Nilai Tugas, Nilai UTS, Nilai UAS

Menghitung nilai akhir berdasarkan bobot: Tugas: 30%, UTS: 35%, UAS: 35%

Menyimpan data ke dalam list.

Menampilkan pertanyaan: Tambah data (y/t)?

Jika user memilih t, program menampilkan tabel seluruh data mahasiswa.
